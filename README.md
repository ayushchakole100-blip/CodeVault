# 🚀 CodeVault

A full-stack Data Structures & Algorithms platform inspired by LeetCode, built to help users practice coding problems, track progress, analyze performance, and receive personalized problem recommendations.

---

## ✨ Features

- 🔐 Secure JWT Authentication
- 👤 User Registration & Login
- 📚 Problem Library
- 💻 Online Code Editor
- ▶️ Run & Submit Code
- 📊 Performance Analytics
- 🎯 Goal Tracking
- 🏆 Leaderboard
- 🤖 Personalized Problem Recommendation Engine
- 📈 Dashboard with Statistics
- 🗄️ MySQL Database
- ⚡ RESTful APIs using Express.js

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ayushchakole100-blip/CodeVault.git
```

```
cd CodeVault
```

---

### Backend

```
cd server
npm install
npm start
```

---

### Frontend

```
cd client
npm install
npm run dev
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the **server** folder.

```
PORT=5000

DB_HOST=localhost

DB_PORT=3306

DB_USER=your_username

DB_PASSWORD=your_password

DB_NAME=codevault

JWT_SECRET=your_secret_key
```
