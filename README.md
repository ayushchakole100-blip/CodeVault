# CodeVault

CodeVault is a full-stack competitive programming analytics and problem recommendation platform.

## Features

- User authentication
- Problem tracking
- Submission analytics
- Topic-wise performance analysis
- Weak topic detection
- DSA-based problem recommendation engine
- Coding streak tracking
- Leaderboard
- User goals

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Algorithm Engine
- C++
- Data Structures and Algorithms
